Introduction to Software Development¶

This ‘book’ is an introduction to the basic principles of software development using Python. This material is targeted toward first-year college students with no programming experience, but that means it should also be accessible to anyone with basic computer skills who wants to learn the basics of computer programming!

This book uses short readings, videos, and interactive review questions to teach the Python programming language. Along the way, we’ll also discuss the importance of testing, debugging strategies, and other topics adjacent to computer science.

If you’re using this book as part of the CSSE 120 course at the Rose-Hulman Institute of Technology, you’ll also complete more rigourous programming exercises. Some of the videos in this book reference the code from these exercises, but the code itself is delivered by a separate mechanism, and therefore does not appear here.

Organization¶

This book is organized into sessions, each corresponding to a single class day, with three sessions a week. Other content that doesn’t quite fit into a particular session is included at the end of the book.

  • Session 1: Introduction to Python
  • Session 10: Sequences

Format¶

Throughout the book you’ll see a variety of multi-media content, including videos, code cells, and interactive review questions.

For example, you might see small Python code snippets:

total = 0
for i in range(5):
    total = total + i
print(total)

Which can optionally be followed by the output of the code.

for i in range(5):
    print('Iteration:', i)

Iteration: 0
Iteration: 1
Iteration: 2
Iteration: 3
Iteration: 4

You may also encounter notes, which highlight key points within the text.

Note

Here is a note!

Finally, this book makes use of interaction review questions via JupyterQuiz so that you can test your knowledge as you work through the content. Most sections will include a few questions relevant to that topic area.
